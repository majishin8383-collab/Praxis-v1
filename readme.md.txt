# Praxis v1

A simple, static “daily check-in + today plan” app.

## Features
- Check-in (mood 0–10 + notes)
- Today Plan (3 priorities + smallest step + time box)
- History log
- Export/Import JSON
- Uses localStorage (private by default)

## Run locally
Just open `index.html` in a browser.

## Deploy (Netlify)
- New site from Git
- Connect your GitHub repo
- Build command: (none)
- Publish directory: `/` (root)

## Deploy (GitHub Pages)
- Settings → Pages
- Source: Deploy from a branch
- Branch: `main` / root
